# Trainsit Application Integration Page

Static Vercel-ready page describing application integration options between the marketing website and app.trainsit.com.

## Files
- `index.html`
- `vercel.json`

## Deploy to Vercel
1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Import the repository into Vercel.
4. Deploy.

No build step is required.
